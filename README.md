# ExWeave —— 从聊天记录到社会关系网络

> 一个 AI 能模仿一个人的语气。但要想模仿她怎么对待不同的人，得先知道她和每个人都什么关系。

---

## 起点

这个项目起初很简单：把某个人的微信聊天记录蒸馏成一个 AI Skill，让模型模仿她的语气说话。

效果不错。上万条消息训练出的模型，用她习惯的 6-7 个字，在她该沉默的时候沉默，在她该爆粗的时候爆粗。

然后一个问题出现了。

你问 AI："你觉得某某怎么样？" AI 只能查一张写死的标签表。它认得这个名字，但不知道这个名字意味着什么——不知道他们互动过多少次、在哪认识的、关系是远是近。

**刻舟求剑。标签是死的，关系是活的。**

## 核心问题

同一个人，在不同人面前说不同的话。

对室友聊到凌晨，对班主任不超过五个字。在项目群里吐槽领导，在班级群里只发"收到"。对着喜欢的人写写删删，对着路人随手一个表情包。

**她的每一句话，都是一次社会关系的应答。** 如果不重建她所处的社会网络，就不可能真正理解她的任何一句话。

## ExWeave 做了什么

把数十份聊天记录——私聊、群聊——全部导入 Neo4j 图数据库，构建出数百人、数万条消息的社会关系网络。然后在对话时，让 AI 去查这张网，而不是翻标签表。

```
                     ┌──────────────────────┐
                     │   社会关系网络        │
                     │   数百人 · 数万条消息  │
                     │   PageRank · Louvain  │
                     └──────────┬───────────┘
                                │
       ┌────────────────────────┼────────────────────────┐
       │                        │                        │
       ▼                        ▼                        ▼
   "某某怎么样"              "某某你认识吗"           "这人谁啊"
   强度 85 · 共同群 3        强度 35 · 共同群 1         网络中无
   互动 500+                 互动 5
       │                        │                        │
       ▼                        ▼                        ▼
   "他啊 烦得要死"            "哦 好像见过"              "不认识"
```

**三层信息融合**：

| 层 | 来源 | 回答 |
|------|------|------|
| 语义记忆 | Neo4j 关键词搜索 | 她说过什么 |
| 社会网络 | PageRank + 关系强度 + 共同群聊 | 她和谁什么关系 |
| 统计约束 | 消息行为分布 | 她怎么说话 |

## 对比

| | 单人 Skill | ExWeave |
|------|-----------|---------|
| 数据 | 1 份私聊 | 数十份聊天（私聊 + 群聊） |
| 人物认知 | 手写标签 | 图数据库动态查询 |
| 语气 | 一刀切 | 随关系强度变化 |
| 扩展 | 每加一人重写配置 | 自动解析，自动建模 |
| 时间维度 | 无 | 关系生命周期 + 时序分析 |

## 快速开始

```bash
pip install neo4j openai flask
export DEEPSEEK_API_KEY=sk-...

python tools/rebuild_all.py
```

## 工具地图

| 你要什么 | 跑这个 |
|---------|--------|
| 谁是我的核心圈 | `graph_analytics.py --pagerank` |
| 社交圈分几层 | `graph_analytics.py --community` |
| 和某人关系到底怎样 | `relationship_analyzer.py` |
| 关系怎么变化的 | `temporal_evolution.py` |
| AI 对话 | `graph_api.py` + Claude Code Skill |

## 对话仿真 API

```
GET /api/graph/smart?query=...     # 语义记忆
GET /api/graph/social?person=...   # 社会网络上下文
```
