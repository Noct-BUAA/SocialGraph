# ExWeave —— 把聊天记录织成一张网，让 AI 真正理解一个人

> *你说了一万句话。每一句我都记得。*
> *但直到我把你身边的所有人都织进同一张网里，我才开始理解你。*

---

## 它始于一个私人的、几乎羞于启齿的念头

把某个人的微信聊天记录喂给 AI，让它模仿她的语气说话。

它确实做到了。上万条消息训练出的模型，能用她惯用的 6-7 个字，在她该沉默的时候沉默，在她该爆粗的时候爆粗。认识她的人试过——分辨不出来。

但很快，一道墙横在了面前。

你问她："你觉得某某怎么样？"

AI 只能回答预设好的标签。它知道一个名字，但它不知道这个人对她意味着什么。它不知道她和他深夜聊过多少次，不知道他们在哪个群里认识，不知道她提起他时语气里的那些微妙变化——从热络到敷衍，从敷衍到沉默。

**就像一个演员背熟了台词，却没读过剧本。她知道说什么，却不知道自己在对谁说话。**

## 一个人的语言，是她所处世界的投影

人在不同的人面前，是说不同的话的。

和室友聊到凌晨三点，句句都是真心话。对着班主任，回复从来没有超过五个字。在项目群里疯狂吐槽领导，转头在班级群里只发"收到"。对喜欢的人小心翼翼删了又改，对无关的人随手回个表情包。

**她的每一句话，都是在回应她与世界的关系。** 如果你不了解这个关系网络，你就不可能真正理解她的任何一句话。

这就是 ExWeave 要做的事。

## 不只是模仿一个人，而是重建她的整个世界

ExWeave 把数十份聊天记录——私聊的、群聊的——全部喂进 Neo4j 图数据库，构建出一张数百人、数万条消息的社会关系网络。

然后，当对话中提到任何一个人时，AI 不再是翻看一张静态的标签清单。它去查这张网：

```
                     ┌──────────────────────┐
                     │   她的社会关系网络     │
                     │   数百人 · 数万条消息  │
                     │   PageRank · Louvain  │
                     └──────────┬───────────┘
                                │
       ┌────────────────────────┼────────────────────────┐
       │                        │                        │
       ▼                        ▼                        ▼
   "那个谁怎么样"            "某某你认识吗"           "这人谁啊"
   （三年老友，互损不倦）     （只是群里见过）          （完全陌生）
       │                        │                        │
       ▼                        ▼                        ▼
   强度 85 · 共同群 3       强度 35 · 共同群 1          网络中查无此人
   互动 500+ 条             互动 5 条
       │                        │                        │
       ▼                        ▼                        ▼
   "他啊 烦得要死"           "哦 好像见过"               "不认识"
   （翻白眼，但语气亲昵）      （耸肩，懒得细想）          （没有假装）
```

**三层记忆，缺一不可：**

| 层 | 回答的问题 | 没有它 |
|------|-----------|--------|
| 语义记忆 | "她说过什么？" | 她像失忆了一样 |
| 社会网络 | "她在谁面前说？" | 她对所有人都一个语气 |
| 统计约束 | "她怎么说？" | 她说话不像她了 |

## 从一个人，到所有人

| | 之前的办法 | ExWeave |
|------|-----------|---------|
| 数据 | 一份私聊，两个人的世界 | **所有聊天**，整个社交宇宙 |
| 知识 | 手写的标签："某某是xx" | **活的图**，点击任何一个节点就能展开一段关系 |
| 语气 | 一刀切 | **因人而异**，亲疏冷热全在关系强度里 |
| 扩展 | 每加一个人，重写一遍 | **扔进去，自动织网** |

## 它还能看见时间

人的关系不是静止的。ExWeave 能告诉你一段关系是怎么变化的——哪个月亲密到凌晨还在聊天，哪个月突然断崖，哪个月又悄悄回暖。

它找到了那些你以为自己忘了、但数据帮你记着的时间点。

## 快速开始

```bash
pip install neo4j openai flask
export DEEPSEEK_API_KEY=sk-...

python tools/rebuild_all.py
```

## 工具地图

| 你要什么 | 跑这个 |
|---------|--------|
| 谁是我网络里最重要的人 | `graph_analytics.py --pagerank` |
| 我的社交圈分几层 | `graph_analytics.py --community` |
| 和某个人的关系到底怎么样 | `relationship_analyzer.py` |
| 关系是怎么一步步变成这样的 | `temporal_evolution.py` |
| 让 AI 用网络感知的方式聊天 | `graph_api.py` + Claude Code Skill |

## 对话仿真 API

```
GET /api/graph/smart?query=...     # 语义记忆
GET /api/graph/social?person=...   # 社会网络上下文
```

两个端点，一个 Claude Code Skill。对话时自动查询，语义记忆和社会关系一起注入回复。
